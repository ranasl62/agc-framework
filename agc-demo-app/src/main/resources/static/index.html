MCP bridge (JSON-RPC)

POST /demo/mcp with tools/list, tools/call, or initialize. Each tools/call uses the same gateway as POST /demo/run.

curl -s -X POST http://localhost:8080/demo/mcp -H 'Content-Type: application/json' -d '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"search","arguments":{"q":"audit"},"context":{"traceId":"mcp-1","correlationId":"mcp-1","tenantId":"demo","principalId":"demo-user","roles":["user"]}}}'
